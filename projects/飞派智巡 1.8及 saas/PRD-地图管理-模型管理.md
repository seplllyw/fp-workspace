# 地图管理 & 模型管理 — 产品需求文档（PRD）

> **面向读者**：前端开发 + 产品 / 设计
> **对应页面**：`map-mgr.html`（同一页面内左右面板切换）
> **需求来源**：`智巡Sssa功能清单0625_副本.xlsx` 智巡 sheet（行号标注于文末）
> **范围说明**：本轮已实现的功能 + 与之相关的 Excel 已规定但未实现的需求（标注部分）
> **约定**：表格中"Excel"指上方功能清单；"本轮新增"指本轮在 UI 中实现但 Excel 未描述的需求

---

## 0. 名词解释

| 术语 | 含义 |
|---|---|
| 标注 | 用户在地图上放置的标记（点 / 线 / 面 / 圆） |
| 模型 | 用户上传到组织的二维 / 三维地图数据资源 |
| 3D 文件夹 | 仅对三维模型生效的二级分类容器（默认 + 用户自定义） |
| 选中 | 单条目标被高亮（用于查看 / 编辑 / 批量操作） |
| 回中 | 把地图飞行到目标所在的中心位置 |
| 默认区域定位 | 同一类型模型的中心点居中显示（飞到可视范围） |
| 显隐 | 目标在地图上是否可见（列表内仍存在） |
| 移除 | 从当前地图视图移除（非删除资源） |
| 选中 / 待确认 | 第一次点击删除按钮进入 armed 状态；视觉上变红 / 描边加粗 |

## 2. 地图管理

### 2.1 顶部搜索栏（Excel R190-195）

6 个 Tab 切换：**地点 / 经纬度 / 标注 / 设备 / 模型 / 照片**

| Tab | 输入框 placeholder | 搜索范围 | 选中后行为 |
|---|---|---|---|
| 地点 | "输入地点名称搜索…" | 模糊 + 联想输入 | 地图自动定位并缩放至目标 |
| 经纬度 | "例如 102.85, 24.88" | 十进制 / 度分秒 自动识别 | 地图打点 |
| 标注 | "搜索标注名称或标签" | 名称 / 备注 / 标签 | 定位并高亮对应标注 |
| 设备 | "搜索设备名称 / SN" | 无人机 / 机场 / 车载 | 定位设备当前位置（在线/离线/飞行中） |
| 模型 | "搜索模型名称或区域" | 三维 / 倾斜摄影 / 点云 | 自动加载并定位模型区域（Loading 态） |
| 照片 | "搜索照片名称或时间" | 巡检 / 航拍 / 问题照片（按 AI 识别筛选） | 定位拍摄点 + 缩略图列表预览 |

**通用规则**（Excel R190 备注）：
1. 支持模糊搜索 + 联想输入
2. 输入关键词后下拉展示候选列表
3. 点击结果后地图自动定位 + 缩放
4. 无结果时提示"未找到相关地点"（或对应类型）
5. 已删除 / 无权限的资源不展示
6. 点击候选列表外区域 → 关闭下拉

### 2.2 左侧标注列表面板

#### 2.2.1 顶部操作栏
- 标题"地图标注" + 信息图标 `i`（hover 显示标注管理说明）
- 5 个图标按钮（从左到右）：
  - 筛选（filter 图标）：弹筛选面板
  - 搜索（search 图标）：显示 / 隐藏搜索输入框
  - 统计（bar-chart 图标）：标注数据统计
  - 编辑（square-pen 图标）：进入编辑模式
  - 新建文件夹（folder-plus 图标）：弹窗输入新文件夹名

#### 2.2.2 文件夹结构（Excel R1152, R1164, R345, R347）

固定显示 2 个系统文件夹 + 用户自定义文件夹：

| 文件夹 | 可删除 | 可重命名 | 内容 |
|---|---|---|---|
| 默认文件夹 | ❌ | ❌ | 未分类标注 |
| App 上共享 | ❌ | ❌ | 跨端共享的标注（链接图标：跳到 App 查看） |
| 自定义文件夹 | ✅（必须空） | ✅ | 拖入或新建的标注 |

- 最多 5 层目录
- 每行：chevron（折叠 / 展开）+ 文件夹图标 + 名称 + 数量徽章 + 显隐按钮 + （App 共享：外部链接图标；自定义：删除按钮）
- 显隐按钮：点击切换该文件夹下所有标注在地图上的可见性（视觉上变灰）
- 自定义文件夹删除前必须为空，否则弹 toast"文件夹内还有标注，请先移走或删除"
- **支持拖拽调整层级**（Excel R347）—— 本轮未实现
- 快捷键折叠 / 展开所有文件夹 —— 本轮未实现

#### 2.2.3 标注列表项
- 左侧：类型 icon（点 / 线 / 面 / 圆各一个 Lucide 图标）
- 中间：标注名称
- 右侧（hover 时出现）：居中按钮、删除按钮
- 选中状态：背景高亮（蓝底）

### 2.3 标注类型（Excel R196-276）

#### 2.3.1 单点标注

**绘制流程**（Excel R197, R1168）：
1. 点击工具条"点标注"按钮进入绘制模式
2. 鼠标移动到地图 → cursor 变 crosshair
3. 点击地图落点 → 自动保存，弹出气泡
4. **双击地图** → 选中并打开详情面板，结束绘制

**默认名称**（Excel R199, R1169）：
- 默认 `当前用户名称-序号`（例："张三-1"）
- 用户可手动修改，名称**不能为空**

**可设置字段**（Excel R1169, R200-208, R169-170, R203-208, R200-208）：
- 名称
- 颜色（预设色板，实时预览）
- 图标（8 种 Lucide 图形：diamond / flame / person-standing / car / triangle-alert / check / x / plus）
- 字体大小（小 / 中 / 大）
- 经纬度显示模式（十进制 / 度分秒）
- 经纬度输入（可手动调整点位位置）
- 简介 / 备注
- 图片（单张，jpg / png）
- 绘制者信息（自动填充当前用户）

**测量数据板块**（Excel R203-208）：
- 标题 hover 提示"只能测量出当前视窗范围内的数据"
- 经纬度 / ASL / AGL 高度实时显示

**删除**（Excel R209, R210）：
- **方式 1**：鼠标选中后按 `Delete` 键 → 直接删除，**无二次确认**
- **方式 2**：右键菜单"删除标注" → **第一次点击选中，第二次点击删除，不弹窗**

**右键菜单**（Excel R210, R1180）：
- 删除标注
- 选择设备飞向（设备调度）
- 全部返航（设备调度）
- **飞向这里**（本轮新增）— 直接 `flyTo` 该点位置

**分享**（Excel R1170）：
- 二维码分享：扫码后 H5 显示标注信息 + 导航（免登录）

**回中**：hover 状态点居中按钮 → 地图飞行至该点

#### 2.3.2 线状标注

**绘制流程**（Excel R214, R1171）：
1. 工具条选"线标注" → cursor 变 crosshair
2. 左键依次添加节点
3. **双击地图结束** → 自动保存
4. 鼠标移动有橡皮筋预览线段

**自动计算字段**（Excel R227, R228, R229, R230, R214, R226）：
- 水平距离（米，**每段自动计算**）
- 直线距离（首末点距离）
- 高度差（需要高程数据，本轮预估 ±10m）
- 简介 / 备注
- 图片
- 绘制者

**可设置字段**：与单点相同（颜色 / 线段形态 / 字体大小 / 经纬度模式 / 简介 / 图片 / 绘制者）

**线段中点距离标签**（UI 行为）：
- 地图上每段线段中点显示距离标签（如 "30.5 m"），鼠标拖动 / 缩放地图时实时更新

**删除**（Excel R232, R233）：
- 同单点：`Delete` 键 / 右键两次点击
- 不弹任何二次确认

**右键菜单**（Excel R233, R210 备注）：
- 删除标注
- 转换为测区

#### 2.3.3 面状标注

**绘制流程**（Excel R236, R1174）：
1. 工具条选"面标注" → cursor 变 crosshair
2. 左键添加顶点
3. **双击闭合** → 自动保存
4. 鼠标移动有橡皮筋预览多边形

**自动计算字段**（Excel R249-253, R236）：
- 水平距离（米）
- 面积（平方米，**多边形自动计算**）
- 周长
- 简介 / 备注
- 图片
- 绘制者

**可设置字段**：与单点相同

**中心标签**（UI 行为）：
- 地图上面状标注的几何中心显示名称 + 面积（如 `北大村特大桥 1.2 m²`）

**删除**（Excel R255, R256）：
- 同单点：`Delete` 键 / 右键两次点击

**右键菜单**（Excel R256, R210 备注）：
- 删除标注
- 转换为测区
- **巡逻航线**（本轮新增，hover 高亮）— 标记为巡逻航线
- 转换为巡逻区域

#### 2.3.4 圆形标注

**绘制流程**（Excel R260, R1177）：
1. 工具条选"圆形标注" → cursor 变 crosshair
2. **第一次点击确定圆心** → 显示中心点 marker
3. **第二次点击确定半径**（鼠标位置为圆周点）→ 实时显示半径长度（米）→ 自动保存

**自动计算字段**（Excel R269-270, R260）：
- 圆心经纬度
- 水平半径
- 面积（平方米）
- 周长
- 简介 / 备注
- 图片
- 绘制者

**可设置字段**（Excel R264-265, R263）：
- 颜色
- 透明面开关（只显示圆线 vs 显示填充）
- 字体大小
- 经纬度模式
- 简介 / 图片 / 绘制者

**删除**（Excel R275, R276）：
- 同单点：`Delete` 键 / 右键两次点击

**右键菜单**（Excel R276）：
- 删除标注（圆形**无**转换菜单项）

### 2.4 地图右侧工具条（Excel R277-292, R309-310）

完全照搬 `index.html` 右侧工具条布局。从上到下：

| 序号 | 按钮 | 行为 | 备注 |
|---|---|---|---|
| 1 | 放大（zoom-in） | `map.zoomIn(1)` | 同步更新 zoom 数字 |
| 2 | zoom 数字 | 显示当前层级 | 例"14" |
| 3 | 缩小（zoom-out） | `map.zoomOut(1)` | |
| 4 | divider | 视觉分隔 | |
| 5 | 复位视野（house） | `flyTo` 初始中心 [24.880, 102.929] / zoom 11 | |
| 6 | 全屏（maximize） | 进入 / 退出 `.main` 区域全屏（兼容 webkit/moz） | |
| 7 | 测距（ruler） | 测距模式 | 详见 §2.4.1 |
| 8 | divider | | |
| 9 | 点标注（map-pin） | 点绘制模式 | 详见 §2.3.1 |
| 10 | 线标注（minus） | 线绘制模式 | 详见 §2.3.2 |
| 11 | 面标注（hexagon） | 面绘制模式 | 详见 §2.3.3 |
| 12 | 圆形标注（circle） | 圆绘制模式 | 详见 §2.3.4 |
| 13 | 键盘操作（keyboard） | 弹出快捷键说明卡片 | |

**激活规则**：
- 任意时刻**仅一个工具激活**（4 个绘制工具 + 测距互斥）
- 激活时按钮蓝色高亮 + 左侧蓝色描边条
- 再次点击同一按钮 → 退出激活

**Tooltip**：所有工具按钮 hover 显示中文 tip（"放大 / 缩小 / 复位视野 / 全屏 / 测距 / 点标注 / 线标注 / 面标注 / 圆形标注"）

**z-index 规则**：
- 工具条 z-index 必须高于右侧信息卡片，否则工具条被遮挡无法点击

#### 2.4.1 测距工具（Excel R309）

- 激活后 cursor 变 crosshair
- 每次点击地图添加一个测量点（带红色圆点 marker）
- ≥2 个点后自动画红色虚线段
- 实时计算并显示在工具条 tip 上：`总长：1234.5m`
- **双击地图** → 结束测距
- 再次点击"测距"按钮 → 退出测距并清除画线
- 与 4 个绘制工具互斥

#### 2.4.2 键盘快捷键（Excel R308）

| 键位 | 行为 |
|---|---|
| `Esc` | 取消当前绘制 / 退出测距 / 取消选中 |
| `Delete` / `Backspace` | 删除选中的标注（**无弹窗**） |
| `↑` / `↓` / `←` / `→` | 平移地图（每次 80 像素，动画过渡） |
| `+` / `=` | 放大地图（zoomIn 1 级） |
| `-` / `_` | 缩小地图（zoomOut 1 级） |
| `R` / `0` | 复位视野（`flyTo` 初始中心 [24.880, 102.929] / zoom 14） |
| `Q` | 环顾左（heading -15°） |
| `E` | 环顾右（heading +15°） |
| `W` | 视角上仰（pitch -5°） |
| `S` | 视角下俯（pitch +5°） |
| `1` / `2` / `3` / `4` / `5` | 工具条前 5 个按钮（放大 / 缩小 / 测距 / 点标注 / 圆标注） |

**通用规则**：
- 焦点在 `<input>` / `<textarea>` / `contenteditable` 元素上时，**不触发**快捷键
- 按住 `Ctrl` / `Meta` / `Alt` 时不触发（避免劫持浏览器原生快捷键）
- 视角调整（Q/E/W/S）通过旋转 tile pane 模拟（Leaflet 2D 卫星图原生不支持 bearing）

## 3. 模型管理（Excel R147-173, R1181-1198）

### 3.1 顶部工具条
- 标题"地图模型"
- 信息图标 `i`（hover 显示模型管理说明）
- 刷新按钮（rotate-cw 图标）

### 3.2 文件夹栏（仅 3D 模型）

```
[+ 新建文件夹]                    当前：<活动文件夹名>
```

- 左侧虚线"新建文件夹"按钮：点击后弹 `prompt`，输入名称（默认"新文件夹"）→ 创建后**自动选中**该文件夹作为活动文件夹
- 右侧"当前：xxx"实时显示活动文件夹（**新模型落点**）
- 活动文件夹规则：3D 模型新放置时落入此文件夹（导入功能未做）

**3D 文件夹列表**（每个文件夹一行）：

| 元素 | 行为 |
|---|---|
| chevron 折叠 | 点击切换展开 / 收起 |
| 文件夹图标 + 名称 | 点击 → 选中为活动文件夹（蓝色高亮） |
| 数量徽章 | 该文件夹下模型数 |
| X 按钮（仅自定义） | 删除文件夹（**必须为空才允许删除**） |

- 默认文件夹：系统自带，不可删除
- 用户新建文件夹：可删除
- 状态持久化：页面刷新后保持展开 / 收起

### 3.3 2D 模型 section（Excel R164-172）

- **不走文件夹结构**，直接以独立 section 形式列在 3D 下面
- 标题"二维模型" + 数量徽章 + 默认区域定位按钮
- 工具条：
  - 资产筛选下拉
  - 状态筛选下拉（建图中 / 建图完成 / 建图失败 / 建图取消）
  - 名称搜索框
- 列表展示 2D 模型卡片

### 3.4 模型卡片（Excel R1182, R1197）

每张卡片显示：

- 多选 checkbox（左侧）
- 模型名称
- 元信息：`子类型 · 采集时间 · 文件大小 MB`
- 资产标签（Lucide tag 图标 + 名称，如"桥梁"）
- 桩号标签（关联桩号时显示）
- 操作按钮（hover 出现）：
  - **回中**（crosshair）：地图 `flyTo` 该模型位置，zoom 不低于 17
  - **显隐**（eye / eye-off）：切换在地图上可见
  - **删除**（trash / trash-2）：**两次点击模式**，详见 §3.5
- 拖拽排序（按住拖动到指定位置）

### 3.5 删除模式（两次点击，无弹窗，本轮新增规则）

- 列表卡片：
  - **第一次点击删除按钮**：卡片变红（红色背景 + 红色描边 + 名称变红 + 删除按钮实心红色），删除按钮上方出现"再次点击删除"气泡提示
  - **第二次点击删除按钮**：真正从列表和地图移除
  - 如果鼠标移出卡片范围（mouseleave）→ 自动取消 armed 状态
- 地图上模型 marker 的删除（Excel R163 / R172）：**弹窗确认**"您确定要移除所选的模型吗？移除的模型将不会在地图上加载"（这是**移除**，不是删除资源，列表展示不变）

### 3.6 选中 / 详情条

- 单击卡片（不在操作按钮上）→ 选中（蓝色高亮），用于查看详情
- 选中后顶部出现选中详情条：
  - 模型名称 + 关闭按钮
  - 字段：类型（3D / 2D 图标）/ 子类型 / 状态 / 资产 / 格式 / 大小 / 采集时间 / 上传者 / 桩号
  - 点击关闭 → 取消选中

### 3.7 批量操作条

- 多选 checkbox：累积到批量操作条
- 批量操作条（顶部出现，仅当选中 ≥1 时）：
  - "已选 X"
  - 批量显示（vis-on）
  - 批量隐藏（vis-off）
  - 批量回中（center）：依次 `flyTo` 选中项
  - 批量删除（del，**两次点击模式**，与单条一致**）
  - 清空选择（X 按钮）

### 3.8 用量标识（Excel R151）

- 顶部"云端建图：已用 X 张 / 共 Y 张" + 进度条
- 数值来源：组织套餐设定
- 包含**模型重建**与**实时建图**

### 3.9 默认区域定位（Excel R155, R167）

- 3D / 2D section 标题旁有 crosshair 按钮
- 点击后：
  - 若该类型下有可见模型 → 地图 `flyToBounds` 所有可见模型的范围（padding 60px, maxZoom 17）
  - 若该类型下无可见模型 → `flyTo` 地图中心，zoom 14，弹 toast"该分类暂无可见模型"
- 完成后弹 toast"三维模型已定位到默认区域" / "二维模型..."

### 3.10 模型操作状态（Excel R156, R160, R163, R168, R169, R172）

- **默认显示**：所有模型默认在地图上可见
- **隐藏**：点击卡片 eye-off 按钮或地图上 marker → 地图上不显示，列表保留
- **移除**（不是删除）：地图上 marker 的删除按钮 → 弹窗确认 → 仅从地图视图移除，**列表展示不变**（Excel R163 备注："此处不是删除模型，只是在地图上不展示，列表展示不变"）
- **删除**（列表卡片）：两次点击 → 真正从列表和地图移除

## 4. 全局交互规则

### 4.1 互斥规则
- 4 个绘制工具 + 测距：任意时刻仅一个激活
- 激活时蓝色高亮 + 左侧蓝色描边条
- 再次点击同一按钮 → 退出

### 4.2 标注的两步删除

| 场景 | 第一次行为 | 第二次行为 | 弹窗 |
|---|---|---|---|
| 单 / 线 / 面 / 圆 标注右键"删除" | 选中（armed） | 真正删除 | ❌ |
| 单 / 线 / 面 / 圆 标注按 `Delete` 键 | 直接删除 | — | ❌ |
| 3D / 2D 模型列表卡片删除按钮 | 卡片变红 + 气泡"再次点击删除" | 真正删除 | ❌ |
| 3D / 2D 模型地图 marker 删除 | 弹窗确认 | 确认后移除（不删资源） | ✅ |

### 4.3 取消规则
- `ESC` → 取消当前绘制 / 退出测距
- 鼠标移出模型面板 → 取消 armed 状态
- 双击地图 → 结束线 / 面 / 圆绘制（自动保存）/ 结束测距

### 4.4 地图事件优先级
- 激活绘制模式时：地图 click 用于绘制节点
- 激活测距模式时：地图 click 用于添加测点
- 默认状态：地图 click 用于其他逻辑（标注选中 / popup 打开）

### 4.5 资源为空时
- 文件夹展开后无内容 → "暂无标注" / "此文件夹下暂无模型"
- 整列表面板为空 → "暂无标注"
- 默认区域定位无可见目标 → toast 提示
