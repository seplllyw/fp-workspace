## 一、整体态势 — 地图右侧工具栏

就是地图右上角那个竖着的工具栏，原型里面有。目前 `index.html` 已经有了。

**要做的事：**

1. 按钮从上到下依次是：放大、当前层级数字、缩小、分割线、复位视野、全屏、分割线、测距、分割线、点标注、线标注、面标注、圆标注、分割线、键盘说明
2. 按钮有三种状态：默认、hover、激活。激活态是蓝底白图标，左边有条蓝色竖线
3. 测距和四个标注工具（点/线/面/圆）互斥，同一时间只能激活一个。再点一次取消激活
4. 每个按钮 hover 时左边浮出中文提示（放大、缩小、复位视野、全屏、测距、点标注、线标注、面标注、圆形标注、键盘说明）
5. 全屏按钮点了之后整个 main 区域全屏，再点退出
6. 键盘说明按钮点了弹出一个快捷键卡片

---

## 二、整体态势 — 地图左下角工具栏

地图左下角，横向的一条。

**要做的事：**

1. 从左到右：默认区域定位按钮、图层管理按钮、比例尺、鼠标坐标
2. 默认区域定位：点了之后地图回到默认视野范围
3. 图层管理：点了之后弹出图层面板，可以开关各个图层（资产、设备、正射影像、限制区、禁飞区、禁降区、手动标记、3D模型、倾斜摄影）
4. 比例尺：跟着地图缩放级别实时变
5. 鼠标坐标：鼠标在地图上移动时实时显示经纬度，格式像 `102.85°E, 24.88°N`
6. 图层面板可以折叠成一个小胶囊，点了再展开，里面有搜索框可以搜资产和正射影像

## 四、模型管理页面

这个页面目前没有地图，是一个纯文件管理界面。左右布局：左边资源目录树，右边文件网格。

**要做的事：**

1. 把地图加进去。在右侧区域嵌入 Leaflet 地图，底图用 Esri 卫星图
2. 右侧工具栏加上，和整体态势一样。但标注工具按钮在模型管理页面点了不用响应（或者弹个提示说这里不支持标注）
3. 左下角工具栏和图层面板加上，和整体态势一样
4. 地图上要显示模型标记点。不同资产类型用不同颜色区分：
   - 桥梁紫色、边坡橙色、隧道青色、交安设施红色、地质灾害深红、服务设施绿色、避险车道橙色
5. 点击模型卡片 → 地图飞到那个模型的位置
6. 点击地图上的模型标记 → 左侧面板选中对应的模型

## 五、图层面板

三个页面共用，左下角那个。

**要做的事：**

1. 分三个区：业务（资产、设备、正射影像）、空域安全（限制区、禁飞区、禁降区、手动标记）、模型（3D模型、倾斜摄影、透明度滑块）
2. 每个图层项有一个开关，点整行就能切换
3. 资产和正射影像是可展开的，点开有搜索框和列表，展开一个的时候另一个自动收起
4. 面板可以折叠成一个小横条，再点展开
5. 面板里的操作不要触发地图的点击和拖拽

---

## 六、地图上的区域涂层

地图上除了标记点之外，还需要用半透明的色块来表示不同区域的覆盖范围，和图层面板的开关联动。

**要做的事：**

1. 空域安全那几个图层（限制区、禁飞区、禁降区、手动标记），开了之后地图上要显示对应颜色的半透明涂层区域，不是点，是一块一块的面
2. 正射影像图层开了之后，地图上要显示对应的影像覆盖范围
3. 涂层的颜色和透明度参考图层面板里的颜色圆点
4. 图层关了涂层就消失
