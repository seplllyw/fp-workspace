# FP Workspace 项目理解

> 2026-07-07 · AI Workspace Architect (Claude)

---

## 一、这是什么

飞派（FP）产品团队的 AI 协同工作站。用户是无人机行业产品经理，10条业务线：飞派云平台、智巡、卫瞳、消防、数据资产、AI算法平台、智能调度、地图能力、BI、低空经济。

核心思路：让多个 AI 模型（Claude/DeepSeek/MiniMax）在这个 Workspace 里长期协同，知识和产出能复用，不每次从零开始。

---

## 二、目录怎么用

```
FP-Workspace/
├── CLAUDE.md               ← AI 进入先读这个（宪法）
├── README.md               ← 人类看的
│
├── system/                 ← AI 系统本身（角色定义、路由规则、Skill、Prompt、模板）
├── knowledge/              ← 永久知识库（术语、业务域、产品方法论、AI经验）
├── projects/               ← 实际项目代码（不可随意改）
├── inbox/                  ← 临时资料入口
├── outputs/                ← AI 产出归档
├── archive/                ← 历史归档
└── screenshots/            ← 截图暂存
```

## 四、关键设计约定

1. **三个地图页面共用同一套工具栏** — 样式、图标、交互逻辑一致。index.html 是参考基准
2. **CSS 变量体系不同** — index.html 用 `--accent`/`--text-primary`/`--surface-hover`，dashboard 和 map-mgr 用 `--primary`/`--text`/`--primary-soft-2`。值基本对应，名字不同
3. **标注绘制行为不同** — index.html 和 map-mgr 绘完弹出详情面板让用户填内容，dashboard 直接画
4. **model-mgr.html 没有地图** — 它是纯文件管理界面。如果要加地图，需要从零接入 Leaflet + 工具栏 + 图层面板
5. **所有页面无后端** — 纯前端，刷新即重置。Leaflet 加载 Esri 卫星底图
6. **图标用 Lucide Icons** — CDN 引入，所有页面统一
