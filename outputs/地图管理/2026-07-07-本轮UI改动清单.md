## 1. 整体态势(`index.html`)

### 1.1 地图右上角 · 放大缩小工具栏

- 位置:`.main`(或地图容器)右上角,贴边 16px
- 形态:竖向白色浮卡,圆角 6px,投影 `--shadow-md`
- 内容(自上而下,**未激活**态):
  1. 🔍+ 放大按钮(`zoom-in` Lucide 图标)
  2. zoom 数字(当前层级,如 `14`,字号 12px)
  3. 🔍- 缩小按钮(`zoom-out` Lucide 图标)
  4. divider(横线 1px)
  5. 🏠 复位视野(`house`,`flyTo` 初始中心)
  6. ⛶ 全屏(`maximize`,进/退 main 区域全屏)
  7. divider
  8. 📏 测距(`ruler`,激活互斥)
  9. divider
  10. 📍 点标注(`map-pin`)
  11. ➖ 线标注(`minus`)
  12. ⬡ 面标注(`hexagon`)
  13. ⭕ 圆标注(`circle`)
  14. ⌨ 键盘说明(`keyboard`,弹快捷键卡片)
- 状态:
  - 默认:浅灰图标 `#475569`,hover 蓝 `#2b7fff` + 浅蓝底 `rgba(43,127,255,.06)`
  - 激活:蓝底 `#2b7fff` + 白图标 + 左侧 2px 蓝色描边条
  - 仅一个工具可激活(测距 / 4 个绘制互斥)
- hover tooltip:中文(放大 / 缩小 / 复位视野 / 全屏 / 测距 / 点标注 / 线标注 / 面标注 / 圆形标注)
- z-index > 右侧信息卡片

### 1.2 地图左下角 · 工具栏

- 位置:`.main` 左下角,贴边 16px
- 形态:横向白色浮卡,圆角 6px,投影 `--shadow-md`
- 内容(自左向右):
  - 🎯 默认区域定位(`crosshair`,`flyToBounds` 当前可见模型范围)
  - 📂 图层管理(`layers`,弹图层显隐面板)
  - 📊 比例尺(`ruler`,实时显示当前比例尺)
  - 鼠标坐标显示(经纬度,实时跟随鼠标,如 `102.85°E, 24.88°N`)
- 状态:
  - 默认:浅灰,hover 蓝
  - hover 显示坐标 tooltip
