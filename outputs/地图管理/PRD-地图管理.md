## 一、背景

无人机巡检业务在地图上的标注、测量、检索是核心操作场景。运营人员需要在地图上快速创建点位标注、测量距离、管理标注文件夹，并在模型管理模式下维护 3D/2D 模型资产。当前版本将"地图标注"与"模型管理"集成在同一页面，通过左侧面板切换。

## 三、功能模块

### 3.1 地图标注面板（左侧 280px）

#### 3.1.1 面板工具栏

| 按钮 | 图标 | 功能 |
|------|------|------|
| 筛选 | filter | 点击弹 toast：演示版未启用 |
| 搜索 | search | 展开/收起面板内置搜索框 |
| 统计 | bar-chart-2 | 点击弹 toast：演示版未启用 |
| 编辑 | square-pen | 点击弹 toast：批量编辑未启用 |
| 新建文件夹 | folder-plus | 弹出 prompt 输入框创建新文件夹 |

**交互规则**：
- 搜索按钮为 toggle 行为：点击展开搜索框并自动 focus，再次点击收起
- 搜索框输入内容后实时过滤标注列表（按名称模糊匹配）
- 新建文件夹弹出浏览器原生 `prompt`，名称最长 20 字符，不可重名，不可为空

#### 3.1.2 文件夹树

系统预置两个文件夹：

| 文件夹 | 说明 | 特殊操作 |
|--------|------|---------|
| 默认文件夹 | 系统默认，不可删除 | 显隐切换（眼球图标） |
| App 上共享 | 从移动端同步 | 外链图标（App 中查看） |

用户可创建**自定义文件夹**（通过面板工具栏的 + 按钮）。

**每行文件夹操作按钮**：

| 按钮 | 出现条件 | 功能 |
|------|---------|------|
| 显隐切换（眼球） | 默认文件夹 & 自定义文件夹 | 切换该文件夹下所有标注在地图上的显隐 |
| 删除（x） | 仅自定义文件夹 | 二次确认后删除文件夹（文件夹内必须为空） |
| 外链（external-link） | 仅"App 上共享" | 提示"请在 App 中查看共享标注" |

**文件夹折叠/展开**：
- 点击文件夹行即可折叠/展开子列表
- 默认："默认文件夹"展开，"App 上共享"折叠
- 折叠状态通过 chevron 图标旋转 -90° 表示

**文件夹选中（活跃文件夹）**：
- 标注创建时自动归入当前活跃文件夹
- 拖拽标注行到其他文件夹可移动（当前版本为演示）

#### 3.1.3 标注列表行

每条标注行显示：

| 元素 | 说明 |
|------|------|
| 类型图标 | 根据标注的 iconType 显示对应 Lucide 图标（diamond / flame / person-standing / car / triangle-alert / check / x / plus） |
| 名称 | 标注名称，超长省略号截断 |
| 操作按钮（hover 出现） | 居中定位 + 删除 |

**交互规则**：
- **hover**：显示操作按钮（居中、删除）
- **点击行**：选中该标注 → 地图飞至标注位置 → 打开右侧详情面板
- **选中态**：行高亮蓝色背景 + 蓝色文字，地图上标注 marker 高亮
- **删除**：两步确认 —— 第一次点击"删除"按钮，行变为红色高亮（armed 态），显示提示"再次点击删除"；第二次点击确认删除。鼠标移出列表区域自动取消 armed 态
- **居中**：地图 flyTo 到标注坐标，缩放级别不低于 17

#### 3.1.4 标注搜索

- 点击面板工具栏搜索按钮展开搜索框
- 输入内容实时过滤标注列表
- 大小写不敏感，按名称模糊匹配
- 过滤后文件夹内标注数量实时更新

### 3.2 地图主区域

#### 3.2.1 底图

- 默认加载 Esri 卫星影像（World Imagery）

#### 3.2.2 顶部搜索栏

| Tab | 图标 | 搜索目标 |
|-----|------|---------|
| 地点 | map-pin | 地点名称模糊搜索 |
| 经纬度 | locate | 输入经纬度坐标定位 |
| 标注 | bookmark | 搜索已创建的标注 |

- 结果列表最大高度 280px，超出滚动
- 每条结果显示图标 + 标题 + 副标题 + 标签
- 搜索结果条目可点击，点击后飞至对应位置

### 3.3 标注绘制工具

#### 3.3.1 点标注

**交互流程**：
1. 点击工具栏"点标注"按钮（或按数字键 4）
2. 光标变为十字准星，底部通知提示"点击放置标注点"
3. 在地图上单击：自动创建标注点并打开右侧详情面板
4. 按 Esc 或再次点击工具栏按钮取消

**创建行为**：
- 默认名称："李亚伟-单点-N"（N 自增）
- 默认颜色：蓝色（#2b7fff）
- 默认图标：pin（📍）
- 默认字号：14px
- 坐标模式：十进制
- 自动标记创建人（李亚伟）和创建时间

#### 3.3.2 线标注

**交互流程**：
1. 点击工具栏"线标注"按钮
2. 在地图上点击添加节点（每个节点显示蓝色圆点标记）
3. 节点 ≥ 2 时显示虚线预览
4. 双击地图完成绘制
5. 弹出 prompt 输入标注名称
6. 完成后保存并显示线的实际长度

**计算**：
- 水平距离 = 各段距离之和
- 直线距离 = 首尾节点距离
- 高程差 = 模拟随机值（演示用）

#### 3.3.3 面标注

**交互流程**：
1. 点击工具栏"面标注"按钮
2. 在地图上点击添加顶点（每个顶点显示圆点）
3. 顶点 ≥ 3 时显示虚线预览多边形
4. 双击地图闭合多边形
5. 弹出 prompt 输入标注名称
6. 完成后保存并显示面积和周长

**计算**：
- 面积：使用球面多边形面积公式
- 周长：各边距离之和

#### 3.3.4 圆形标注

**交互流程**：
1. 点击工具栏"圆形标注"按钮
2. 第一次点击确定圆心（显示圆点标记）
3. 底部通知提示"再次点击确定半径"
4. 第二次点击确定半径并弹出名称 prompt
5. 完成后在地图上绘制实心圆 + 显示面积和周长
6. 双击地图取消（如果已确定圆心但未点第二下）

## 四、数据模型

### 4.1 标注对象

```
{
  id:          string     // 唯一标识，格式：类型首字母 + 时间戳 + 随机数
  name:        string     // 标注名称
  type:        enum       // point | line | polygon | circle
  lat/lng:     number     // 坐标（点标注）
  points:      [[lat,lng]] // 顶点数组（线/面标注）
  center:      [lat,lng]  // 圆心（圆形标注）
  radius:      number     // 半径米（圆形标注）
  color:       hex        // 标注颜色
  iconType:    string     // 图标类型（仅点标注）
  iconEmoji:   string     // 图标 emoji
  fontSize:    number     // 字号 px
  lineStyle:   enum       // 线型 solid | dashed | dotted
  fillOpacity: number     // 填充透明度 0-1
  coordMode:   enum       // 坐标显示模式 decimal | dms
  description: string     // 描述文本
  imageUrl:    string     // 附件图片 URL
  artist:      string     // 创建人
  createdAt:   datetime   // 创建时间
  folder:      string     // 所属文件夹 ID
  horizontalDistance: number  // 水平距离（线标注）
  linearDistance: number      // 直线距离（线标注）
  area:        number     // 面积 m²（面/圆标注）
  perimeter:   number     // 周长 m（面标注）
  circumference: number   // 周长 m（圆标注）
}
```

### 4.2 文件夹

```
{
  id:   string     // 唯一标识
  name: string     // 文件夹名称（最长 20 字符）
}
```

## 五、异常处理

| 场景 | 处理方式 |
|------|---------|
| 新建文件夹名称为空 | 提示"名称不能为空" |
| 新建文件夹名称重复 | 提示"已存在同名文件夹" |
| 新建文件夹名称过长（>20） | 提示"名称过长（最多 20 个字符）" |
| 删除非空文件夹 | 拒绝，提示"文件夹内还有标注，请先移走或删除" |
| 标注删除 | 两步确认（armed → confirm），鼠标移出自动取消 |
| 线/面标注节点不足 | 双击时节点 < 2（线）或 < 3（面）不触发完成 |
| prompt 取消 | 取消标注创建，清理预览 |
| 搜索无结果 | 显示"无匹配结果" |
