## 一、背景

无人机采集的 3D 模型（BIM、点云、倾斜摄影、OSGB）和二维正射影像需要在统一的地图视图中管理。运营人员需要在地图上查看模型分布、筛选模型类型、管理文件夹归类，并支持单模型详情查看与删除。

模型管理与地图标注共享同一页面（map-mgr.html），通过左侧导航切换。

---

## 二、入口与切换

### 2.1 左侧导航

左侧导航栏提供两个中间 Tab：

| Tab | 图标 | 激活后面板 |
|-----|------|-----------|
| 地图管理 | map | 地图标注面板（280px） |
| 模型管理 | box | 地图模型面板（280px） |

**切换行为**：
- 两个 Tab 互斥，同一时间仅一个激活
- 激活"地图管理"：显示标注面板，隐藏模型面板
- 激活"模型管理"：显示模型面板，隐藏标注面板
- 切换时地图底图不变，顶部搜索栏隐藏（模型管理模式下）

## 四、功能模块

### 4.1 面板工具栏

| 元素 | 说明 |
|------|------|
| 标题"地图模型" | 固定显示 |
| ⓘ 信息图标 | hover 显示"模型管理说明" |
| 🔄 刷新按钮 | 点击刷新面板列表和地图覆盖物 |

**刷新行为**：
- 重新渲染面板列表
- 重新渲染地图上的模型标记
- 更新底部状态栏
- 弹出通知"已刷新"

---

### 4.2 模型选中详情条

当用户点击某个模型卡片时，在面板顶部出现蓝色高亮信息条。

**显示内容**：

| 字段 | 说明 |
|------|------|
| 模型名称 | 加粗显示，前带图标（3D → box / 2D → image） |
| 类型 | 3D 子类型（BIM / 点云 / 倾斜摄影 / OSGB）或 2D 状态 |
| 状态 | 已完成 / 处理中 / 失败 |
| 资产 | 所属资产类型 |
| 格式 + 大小 | 如 "OSGB · 245 MB" |
| 采集时间 | 数据采集日期 |
| 上传人 | 上传用户名 |
| 关联桩号 | 如有关联桩号则显示 |

**操作**：
- 右上角 x 按钮：取消选中，关闭详情条

**交互规则**：
- 点击任意模型卡片 → 该卡片高亮 + 详情条出现
- 选中为单选：点击新卡片会替换旧选中
- 关闭详情条 → 所有卡片取消高亮

---

### 4.3 3D 模型文件夹

#### 4.3.1 文件夹操作栏

| 元素 | 功能 |
|------|------|
| [+ 新建文件夹] 按钮 | 弹出 prompt 创建新 3D 模型文件夹 |
| 当前文件夹指示器 | 显示"当前：xxx"，新建模型将归入此文件夹 |

**新建文件夹交互**：
- 点击按钮 → 弹出 `prompt("请输入文件夹名称", "新文件夹")`
- 名称为空 → 取消创建
- 创建后自动设为当前活跃文件夹（activeModelFolder3d）

#### 4.3.2 文件夹列表

每个文件夹行显示：

| 元素 | 说明 |
|------|------|
| chevron 折叠图标 | 点击折叠/展开子列表 |
| 文件夹图标 | 蓝色文件夹 icon |
| 文件夹名称 | 可自定义 |
| 模型数量 | 圆角数字徽标，如 "(3)" |
| 删除按钮（x） | 仅自定义文件夹显示，默认文件夹不可删除 |

**文件夹行交互**：

| 操作 | 行为 |
|------|------|
| 点击文件夹行 | 选中该文件夹为活跃文件夹（activeModelFolder3d），行高亮蓝色背景 |
| 点击 chevron | 折叠/展开文件夹内的模型列表 |
| 点击 x 删除 | 删除文件夹（当前版本文件夹内模型自动保留，移至默认文件夹） |
| 活跃文件夹 | 蓝色背景高亮，新建模型将归入此文件夹 |

**文件夹拖拽（当前版本）**：
- 模型卡片支持拖拽排序（drag & drop）
- 拖拽时卡片半透明（opacity 0.4）
- 拖过其他卡片时显示蓝色插入线（上方/下方）
- 放下后重新排序并重新渲染

---

### 4.4 模型卡片

每张模型卡片显示以下信息：

| 元素 | 说明 |
|------|------|
| 模型名称 | 加粗 12px，超长省略号 |
| 元信息 | 子类型 + 采集时间 + 文件大小（如"倾斜摄影 · 2026-06-15 · 245 MB"） |
| 资产标签 | 蓝色小标签显示资产类型（桥梁/边坡/隧道/交安设施/地质灾害/服务设施/避险车道） |
| 关联桩号 | 如有链状桩号则显示（棕色加粗） |
| 操作按钮 | 显隐切换 + 删除 |

**模型卡片操作**：

| 按钮 | 图标 | 功能 |
|------|------|------|
| 显隐 | eye / eye-off | 切换该模型在地图上的显示/隐藏 |
| 删除 | minus-circle / trash-2 | 从列表中移除模型 |

**卡片点击行为**：
- 点击卡片 → 选中模型（卡片高亮 + 详情条出现）
- 已选中再点击 → 不变

**删除交互（两步确认）**：
- 第一次点击删除 → 卡片变为红色高亮 armed 态（红色边框 + 红色名称）
- 删除按钮变红底白字，显示"再次点击删除"tooltip
- 第二次点击 → 确认删除，从列表和地图移除
- 鼠标移出面板区域自动取消 armed 态

### 4.5 二维模型分组

二维模型使用折叠分组（section）展示，与 3D 文件夹独立。

#### 4.5.1 分组头部

| 元素 | 说明 |
|------|------|
| chevron 折叠图标 | 点击折叠/展开 |
| 标题"二维模型" | 固定 |
| 模型数量 | 徽标显示 |
| 定位按钮（crosshair） | 点击默认区域定位（地图飞至默认坐标） |

#### 4.5.2 筛选工具条

分组展开后显示筛选行：

| 控件 | 选项 | 影响 |
|------|------|------|
| 资产类型下拉 | 全部资产 / 桥梁 / 边坡 / 隧道 / 交安设施 / 地质灾害 / 服务设施 / 避险车道 | 过滤 3D + 2D |
| 子类型/状态下拉（3D） | 全部子类型 / BIM / 点云 / 倾斜摄影 / OSGB | 过滤 3D |
| 状态下拉（2D） | 全部状态 / 已完成 / 处理中 / 失败 | 过滤 2D |
| 搜索输入框 | 自由文本 | 按名称模糊搜索 |

**筛选规则**：
- 所有筛选条件为 AND 关系（同时满足）
- 资产类型筛选对 3D 和 2D 模型均生效
- 子类型筛选仅对 3D 生效
- 状态筛选仅对 2D 生效
- 搜索输入实时过滤（大小写不敏感）

### 4.7 地图上的模型标记

模型在地图上以 Leaflet marker 形式展示，参照 index.html 的 custom-marker 风格。

**标记样式**：

| 资产类型 | 颜色 | 地图图标 (Lucide) |
|---------|------|------------------|
| 桥梁 (bridge) | #6366f1 紫 | landmark |
| 边坡 (slope) | #f59e0b 琥珀 | triangle-alert |
| 隧道 (tunnel) | #0891b2 青 | minus |
| 交安设施 (traffic-safety) | #ef4444 红 | shield |
| 地质灾害 (geo-hazard) | #dc2626 深红 | alert-triangle |
| 服务设施 (facility) | #16a34a 绿 | building-2 |
| 避险车道 (escape-lane) | #f97316 橙 | corner-up-right |

**标记交互**：
- 圆形标记，白色 2px 边框 + 阴影，28×28px
- **hover**：放大至 1.3x，z-index 提升至最前
- **tooltip**：显示模型名称 + 桩号 + 资产类型
- **点击 marker**：在面板中选中对应模型（卡片高亮 + 详情条出现）+ 通知"已选中：模型名称"
- **显隐控制**：通过卡片上的 eye 按钮切换地图上的显示/隐藏

**模型坐标**：
- 每个模型有独立的 lat/lng 坐标，分散在地图中心（昆明/石林区域）周围
- 不再使用中心网格排列

### 4.8 卡片点击 → 地图定位

点击模型卡片时，除选中模型（卡片高亮 + 详情条）外，地图自动飞至该模型标记位置：

- **飞行动画**：`map.flyTo(markerLatLng, max(当前缩放, 17), duration: 0.6s)`
- **通知反馈**：底部通知"已定位到：模型名称"
- **地图标记联动**：点击卡片时 marker 也会响应（视觉上 marker 保持 hover 态）
- **反向联动**：点击地图上的 marker 也会在面板中选中对应模型

### 4.9 图层面板（工具板 · 左下角）

模型管理页面同时配备图层面板（与整体态势页面一致），浮动在地图左下角：

**面板结构**：
- **业务层** — 资产（可展开列表 + 搜索）、设备（切换开关）、正射影像（可展开列表 + 搜索）
- **空域安全层** — 限制区、禁飞区、禁降区、手动标记（均带切换开关）
- **模型层** — 3D 模型开关、倾斜摄影开关、透明度滑块（0-100%）

**交互规则**（与整体态势一致）：
- 面板可折叠为 40×38px 胶囊
- 资产/正射面板互斥展开（展开一个自动关闭另一个）
- 点击面板外侧自动关闭展开面板
- Escape 关闭展开面板
- 图层项点击整行即可切换开关
- 面板内操作不触发地图交互（mousedown/click/dblclick 全部隔离）
- 搜索输入 150ms 防抖

## 五、数据模型

### 5.1 模型对象

```
{
  id:          string     // 唯一标识
  name:        string     // 模型名称
  type:        enum       // '3d' | '2d'
  subtype:     string     // 3D: bim|pointcloud|oblique|osgb
                          // 2D: ready|pending|failed
  assetType:   string     // 资产类型 ID
  status:      string     // ready | pending | failed（仅 2D 有效）
  fileSize:    number     // 文件大小 MB
  fileFormat:  string     // 文件格式（如 osgb, las, tif）
  collectedAt: string     // 采集日期
  owner:       string     // 上传人
  linkedStake: string     // 关联桩号
  folder:      string     // 所属文件夹 ID（3D）
  visible:     boolean    // 地图显隐
  enabled:     boolean    // 是否启用
  selected:    boolean    // UI 选中态
}
```

### 5.2 资产类型枚举

| ID | 名称 |
|----|------|
| bridge | 桥梁 |
| slope | 边坡 |
| tunnel | 隧道 |
| traffic-safety | 交安设施 |
| geo-hazard | 地质灾害 |
| facility | 服务设施 |
| escape-lane | 避险车道 |

### 5.3 3D 模型子类型

| ID | 名称 |
|----|------|
| bim | BIM |
| pointcloud | 点云 |
| oblique | 倾斜摄影 |
| osgb | OSGB |

### 5.4 2D 模型状态

| ID | 名称 |
|----|------|
| ready | 已完成 |

## 六、交互逻辑汇总

### 6.1 面板切换

```
左侧导航点击"地图管理" → activeMidTab = 'ann'
  → #ann-panel 显示, #model-panel 隐藏
  → 搜索栏显示
  → 底部状态栏切换为标注统计

左侧导航点击"模型管理" → activeMidTab = 'model'
  → #model-panel 显示, #ann-panel 隐藏
  → 搜索栏隐藏
  → 底部状态栏切换为模型统计
```

### 6.2 模型选中链

```
点击卡片 → MOCK_models 设置 selected = true
         → selModel = 该模型
         → 重新渲染面板（详情条 + 卡片高亮）
         → 状态栏更新选中数

点击详情条 x → selModel = null
           → 所有 m.selected = false
           → 重新渲染
```

### 6.3 文件夹操作链

```
点击文件夹行 → activeModelFolder3d = 该文件夹 ID
            → 文件夹行高亮
            → 新建模型栏"当前"文字更新

新建文件夹 → prompt → push 到 modelFolders3d 数组
          → 自动设为 activeModelFolder3d
          → 重新渲染
```

### 6.4 模型搜索/筛选链

```
下拉筛选器 change → 更新对应 filter 变量 → renderModelPanel()
搜索框 input → 更新 filterText → renderModelPanel()
  (保持输入焦点和光标位置)
```

## 七、异常处理

| 场景 | 处理方式 |
|------|---------|
| 文件夹名称为空 | 取消创建，无提示 |
| 删除模型确认 | 两步确认（armed → confirm） |
| 鼠标移出面板 | 自动取消 armed 态 |
| 筛选无结果 | 折叠区域显示"暂无模型" |
| 搜索无结果 | 同上 |
| 地图上模型未显示 | "回中"时提示"该模型在地图上未显示" |
| 删除文件夹 | 文件夹内模型保留（不移除） |

---

## 八、边界条件

- 3D 模型至少有一个默认文件夹（default-3d），不可删除
- 新建文件夹后自动设为活跃
- 模型卡片支持拖拽排序（HTML5 Drag & Drop API）
- 云端建图总量上限 6000 张
- 文件大小以 MB 为单位显示
- 所有面板操作不改变地图底图和缩放
