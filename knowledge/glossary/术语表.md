# 术语表

---

## 二、资产管理术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 资产模版 | Asset Template | 定义一类资产的数据结构（字段+树形结构+编码规则） |
| 在管资产 | Managed Instance | 模版的实例化，代表真实的资产对象 |
| 台账 | Ledger | 资产的基本信息记录 |
| 数字档案 | Digital Archive | 资产的构件分解结构 |
| 树形结构 | Tree Structure | L1-L8 层级化的构件分解体系 |
| 最小维护单元 | Maintenance Unit | 叶子节点实例化后的最小管理单元 |
| 病害 | Defect / Disease | 构件损伤或缺陷 |
| 病害库 | Defect Pool | 所有病害类型的集合 |
| 编码规则 | Encoding Rule | 实体编码 + 构件编码的生成规则 |
| 桩号 | Stake | 公路里程标记，分百米桩和公里桩 |
| 路段 | Section | 桩号的上级组织单元 |
| 构件 | Component | 资产的组成部分（如桥墩、桥面） |

---

## 三、巡检术语

| 术语 | 英文 | 说明 |
|------|------|------|
| 航点 | Waypoint | 无人机巡检航线中的拍摄点位 |
| 架次 | Flight / Mission | 一次巡检飞行任务 |
| 航线 | Route | 预定义的飞行路径 |
| 巡检记录 | Inspection Record | 一次巡检的飞行+病害数据 |

---

## 四、组织术语

| 术语 | 说明 |
|------|------|
| Auth 平台 | 统一身份认证平台，提供组织属性、行业属性 |
| 租户 | Tenant，组织在 SaaS 中的隔离单元 |
| 行业属性 | 组织的行业分类（高速、铁路、水利等），决定桩号编码体系 |

---

## 五、技术术语

| 术语 | 说明 |
|------|------|
| SPA | 单页应用（Single Page Application），无框架依赖的纯前端 |
| API 网关 | 统一 API 入口，负责路由和鉴权 |
| 能力平台 | 底层服务（AI 能力、空间能力、数据资产能力等） |
| MCP | Model Context Protocol，AI 工具调用协议 |
| Skill | 封装了 Prompt + Template + Examples 的可复用 AI 能力单元 |
| Role | 模型角色定义文件 |
| Router | 任务路由系统 |
